# 范围与安全边界

## 仓库定位

本仓库是 OpenClaw 记忆、学习与演化相关工具的**选型与集成参考**，不是一键安装器，也不承诺提供 KAIROS、AutoDream、AutoMemory 等运行时实现。

上游项目、插件接口和 OpenClaw 配置字段可能变化。安装、配置与运行前应以各项目官方文档、当前版本 schema 与本机状态为准。

## 本仓库不做什么

- 不自动安装插件、全局 npm 包或外部 Skill；
- 不自动修改 Gateway 配置或重启服务；
- 不默认启动后台循环、Hub 连接或持续自演化；
- 不替代当前运行时已经启用的记忆、Dream、审计或安全系统。

## 高风险操作

| 操作 | 风险 | 默认策略 |
|---|---|---|
| 安装插件 / Skill | 供应链、权限和依赖风险 | 先临时审计，再确认安装 |
| 修改配置 / 重启 Gateway | 中断会话或路由变化 | 备份、最小 patch、验证后执行 |
| 连接外部 Hub | 日志、元数据或运行信息可能外发 | 先确认数据流与隐私边界 |
| `evolver --loop` 等持续循环 | 长期副作用、资源消耗、行为漂移 | 默认不启用；先 dry-run / review |

## 使用准则

每个上游工具都应明确：它解决什么、不解决什么、是否联网、是否写入本地、是否需要重启、与现有系统是否冲突，以及如何回滚。
